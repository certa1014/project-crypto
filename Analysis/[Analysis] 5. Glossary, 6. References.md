# 5. Glossary

| Term              | Description                      |
| :--- | :--- |
| Player            | 게임을 플레이하며 각종 행동을 수행하는 사용자를 말한다.  |
| NPC               | 플레이어와 상호작용하며 의뢰를 제공하는 캐릭터를 말한다.  |
| Request           | NPC가 플레이어에게 전달하는 의뢰를 말한다.        |
| Ciphertext        | 해독되지 않은 암호문 상태의 문서를 말한다.         |
| Guidebook         | 암호 해독 규칙과 정보를 확인할 수 있는 시스템을 말한다. |
| Modulation        | 해독된 문서 내용을 변조하는 기능을 말한다.         |
| Report            | NPC를 신고하는 기능을 말한다.               |
| Daily Settlement  | 하루 동안의 결과를 정산하는 시스템을 말한다.        |
| Save Data         | 게임 진행 상황이 저장된 데이터를 말한다.          |
| Auto Save         | 하루 종료 시 (일일정산 시) 자동으로 저장하는 시스템을 말한다.  |
| Stage             | 하루 단위로 진행되는 게임 구간을 말한다.          |
| Reputation        | 플레이어의 행동 결과에 따라 변화하는 평가 수치를 말한다. |
| Interaction       | 플레이어와 NPC가 대화하는 시스템을 말한다.        |
| Game Over         | 게임 진행이 종료된 상태를 말한다.              |
